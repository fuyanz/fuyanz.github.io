表单与表单元素

2021前端教案参考
2021前端教案参考
发布日期:   2021-06-02
更新日期:   2021-06-02

表格和表单标签

主要用于显示、展示数据。

<table>
    <tr>
        <td>单元格内的文字</td>
    </tr>
</table>
  1. < table >< /table >定义表格的标签。
  2. < tr >< /tr > 定义表格中的行，必须嵌套在table标签内部。
  3. < td >< /td >定义表格中的单元格，必须嵌套在tr标签中。
  4. td table data，数据单元格的内容。

表头单元格标签

表头单元格位于表格的第一行或第一列，表头单元格里面的文本内容加粗居中显示。

< th >标签用于定于表头部分（table head）

表格属性

属性名 属性值 描述
align left、center、right 规定表哥相对周围元素的对齐方式
border 1或空 默认无边框
cellpadding 像素值 单元边缘与其内容之间的空白，默认1像素。
cellspacing 像素值 规定单元格之间的空白，默认2像素
width 像素或百分比 规定表格的宽度

表格结构标签

为了更好的表示表格的语义，可以将表格氛围表哥头部和表格主体两大部分。

< thead >用于表示表格的头部区域。

​ 内部必须拥有tr标签，一般位于第一行。

< tbody >用于表示表格的主体部分。

合并单元格

跨行合并rowspan=“合并单元格的个数”

最上侧单元格为目标单元格，写合并代码。

跨列合并colspan=”合并单元格的个数”

最左侧单元格为目标单元格，写合并代码。

列表标签

用来布局，根据使用需要分为:无序列表、有序列表、自定义列表。

无序列表

ul标签表示html页面中项目的无序列表，一般会以项目符号呈现列表项，列表项使用< li >标签定义。

<ul>
    <li>列表项1</li>
    <li>列表项2</li>
</ul>

ul中只能嵌套li。

li中可以嵌套其他标签。

有序列表

< ol >

自定义列表

常用语对术语或名词进行解释和描述，定义列表的列表项前没有任何项目符号。

< dl >定义列表< dt >(定义项目/名字) < dd >（描述每一个项目/名字）

<dl>
    <dt>名词1</dt>
    <dd>名词1解释1</dd>
    <dd>名词1解释2</dd>   
</dl>

表单标签

收集用户信息，与用户进行交互。

表单的组成

  1. 表单域
  2. 表单控件（表单元素）
  3. 提示信息

表单域

包含表单元素的区域。< form >会把它范围内的表单元素信息提交给服务器。

< form action=“url地址” method=”提交方式” name=”表单域名” >< /form >

属性
属性 属性值 作用
action url地址 指定接受并处理表单数据的服务器程序的url地址
method get/post 设置表单数据的提交方式
name 名称 指定表单的名称

表单控件

在表单域中定义各种表单元素，这些表单元素是允许用户在表单中输入或选择的内容控件。

input输入表单元素

输入。

input用于收集用户信息。必须属性type。

属性值 描述
button 按钮
checkbox 复选框
file 文件
hidden 隐藏
image 提交图像
password 密码
radio 单选框
reset 重置按钮
submit 提交按钮
text 文本 默认20个字符

属性：

属性 属性值
name
value
checked checked 首次加载时被选中
maxlength 正整数 输入字段的字符最大长度
  1. name和value要给后台人员使用。
  2. name的名字要求单选按钮和复选框要有相同的name值。
  3. checked属性主要针对单选按钮和复选框。
  4. radio（checkbox）如果是一组，必须命名相同的名字。
< label >标签为input元素定义标注。
select下拉表单元素

用于绑定一个表单元素，当点击label标签内的文字的时候，浏览器会自动聚焦到对应的表单元素上，增加用户体验。

label标签的for属性应当与相关元素的id属性相同。

select下拉表单

用于有多个选项让用户选择的时候。

<select>
    <option>1</option>
    <option>2</option>
    <option>3</option>
</select>
  1. select中至少包含一对option。
  2. 在option中定义selected=”selected”，当前选项为默认选中。
textarea文本域元素

用户输入内容较多的情况。

textarea标签用于定义多行文本输入的控件。

cols=“每行中的字符数”

rows=”显示的行数”
