标签

html

双标签

表示整个网页

属性：

  • manifest 定义一个URL，在这个URL上描述了文档的缓存信息（html5）新增

  • xmlns 只有XHTML支持。规定XML的namespace属性

    在HTML5规范中，lang属性是全局属性

    xmlns表示命名空间：namespace

    ​ 我们制作的网页 给不同的用户来浏览，命名空间规定我们的标签代表的含义，必须将命名空间进行统一

标签，元素与属性

元素：网页实际上就是许许多多的html元素构成的文本文件

html元素就是构成HTML文件的基本对象，HTML元素是一个统称

标签：标记html元素

​ 元素是实体，标签是名称（标记）

属性：为html元素提供各种附加信息的就是HTML属性，它总是以“属性名=属性值”的这种名值对的形式出现，而且属性总是在html元素的开始标签中进行定义。

head标签

用于定义文档的头部，是所有头部元素的容器。

head中的元素可以引用脚本，指示浏览器在哪里找到样式表，提供元信息等。

文档的头部描述了文档的各种属性和信息，包括文档的标题，在web中的位置以及和其其他文档的关系等

绝大多数的的文档头部包含的数据都不会真正作为内容显示给读者

  • base标签
    • 为页面上的所有链接规定默认的地址或者目标
    • herf：规定页面中所有相对链接的基准url
    • target：在何处打开页面中的链接（_blank,_parents，_self,_top,_framename)
  • title标签内部放的是网页的名字
    • 里面的内容可以帮我们提高搜索引擎优化（SEO）
  • link 定义文档与外部资源关系
  • style 定义内嵌样式
  • script 引入外部脚本 或定义内嵌脚本
  • meta 提供页面相关的元信息（meta-information），标签位于文档的头部，不包含任何内容
    • content 定义与http-equiv或name属性相关的原信息
    • http-equiv 把content属性关联到HTTP头部（content-type，expires，refresh，set-cookie）
    • name 把content属性关联到一个名称（author，description，keywords（SEO优化等）

字符集

定义了当前文件使用的字库

中文字库：utf-8：国际标准字库 一个汉字3个字节

GBK/gb2312：中国字库，国家标准。一个汉字2个字节。

分情况适用：网页是外文网站用utf-8，有大量中文，需要网页加载速度快用gbk

必须保证设置的字符集和软件保存的字符集类型一致

body标签

作用：body部分是我们的网页主体，给客户浏览的区域

内容：里面放置的是我们html标签的网页解构

html基本语法

标签之间对空格、缩进、换行不敏感，我们的标签嵌套关系敏感

tab：缩进 shift+tab：撤销缩进

空白折叠现象：文本内容区域如果有一个或者多个空格或者缩进或者换行，在浏览器中显示的时候，只显示一个空格

标签分类

html中标签的分类

​ 容器级标签：h1-h6 pre div

​ 文本级标签：p code span

​ 文本级标签一般不能或不建议嵌套容器级标签，但是容器级可以嵌套文本级标签

属性

元素的属性：对元素的描述以及控制信息

属性分类:

  • ​ 必须属性 img中的src alt（图片加载失败提示的内容）

  • ​ 可选属性img 中的 width height

  • 标准（全局）属性 id class style lang

  • 事件属性 onclick，onmouseover，onmouseout

    h系列标签

    给文本添加标题语义用，不能互相嵌套

    默认加粗

    具有换行符

p标签

给文本添加段落语义

具有换行符

p标签可以嵌套在块级元素内

但p标签不能嵌套其他块元素 如div p h

img标签

img标签是一个单标签

img标签的属性：

  • ​ src 添加资源文件的路径
  • ​ alt 图片加载失败后的文本提示
  • ​ title 文本提示信息（鼠标悬停）
  • ​ width height
  • border 设置图片的边框厚度（默认黑色实线)

路径

​ 绝对路径：目标文件所在的完整的路径（带盘符）

​ 相对路径：

  1. ​ /根目录
  2. ​ ./当前目录
  3. ​ ../退出当前目录

a标签

属性：

  • ​ href 设置超链接的地址 #代表连接到当前页面或者某个地址

  • title 文本提示信息

  • rel 规定当前文档与被链接文档之间的关系

  • download 规定被下载的超链接目标

  • media 规定被链接文档是为何种媒介、设备优化的

  • type 规定被连接的文档的mime 类型

  • target 页面的打开方式

    • _self 当前页面中打开新页面 新页面覆盖当前页面
    • _blank 在新的页面中打开 会创建一个新的窗口
    • _parent 页面的父窗口或父框架中打开
    • _top 将页面在整个浏览器窗口打开
    • framename 在指定的框架中打开页面

  • 锚点

    1. 给标签设置id属性(要求id是惟一的)

    2. 通过另一个a标签的href属性去链接这个锚点

      可以连接到当前页面的指定位置

      还可以连接到其他页面的指定部分

div和span标签

div：

容器级标签（division）用来划分独立的逻辑块

是一个容器（大的区域），本身在网页中没有任何默认样式，可以放置任何标签和内容

里面可以存放一些相近，相同功能的标签，div会把页面分割成一个小的区域

span：

文本级的标签，只能放文字图片表单元素等，用来修饰文本的效果（小的跨度范围）

**div作为容器，如果没有内容撑起，或者不设置宽高，是没有默认效果的 **

span标签需要配合css使用

  • ​ 大的内容放div 小的内容放span

  • ​ div盒子会独占一行，span的内容会写在一行

无序列表

给文本添加无序的列表语义，项与项之间没有先后顺序

ul （unordered list）无序列表

li （list item） 列表项

ul和li这两个标签必须同时出现，不能单独书写

ul里可以嵌套一个或者多个li，但ul里面只能放li标签不能放其他内容

li标签是个容器标签可以放置任何内容

默认li是小圆点样式

可以通过type属性修改：disc默认实心圆 circle 空心圆 square 实心方块

有序列表

ol orderred list

li list item

ol里面只能放li

ol元素属性：

​ type 属性设置有序列表的项目符号有且仅有5个

​ A a 1 l i

start 设置列表符号从第几个开始排列，属性值只能是阿拉伯数字

reversed 设置有序列表项目符号的倒序

（这是一个布尔型的属性）

ul ol不能相互使用

定义列表

dl definition list 定义列表

dt definition title 定义标题

dd definition description 定义描述

  • dl 里面嵌套了dt dd。dt，dd是同一级标签dd是对dt的解释说明
  • dl里面只能放置dt，dd。dt和dd都是容器级标签，里面的内容是不限制的
  • dl里面可以放置多组的dt和dd。dt后面的dd可以有多个。这些dd都是在解释上面的dt
  • dt后面可以没有dd，表示没有解释说明
  • 实际工作中，经常将每一组dt和dd单独放在一个dl标签内部

表格基础

table：定义表格容器

tr：table rows 定义行

th：table head 定义表头

td：table dock 定义单元格

三层嵌套关系：table>tr>th|td

最简单的表格要求每一行的单元格个数相同

如果表格有表头的概念：需要将td标签变成th

table标签

属性：border 设置表格的边框

border color 设置边框的颜色

height width：设置表格的宽高

align：设置表格的整体对齐方式

cellspacing：设置边框与边框之间的距离

cellpadding：设置内容与边框之间的距离

background：设置表格的背景图片

bgcolor：设置表格的背景颜色

summary:表格隐藏信息，用来提高SEO

tr,td,th

tr：行语义，一对table标签中有几对tr，就代表这个表格有多少行，属性如下：

​ align 设置的当前单元格中文本的水平对齐方式

bgcolor，background

colspan 列合并，左右合并

rowspan 行合并，上下合并

划分区域

表格标题 caption

表头 thead

主体部分 tbody

设置标题

表格标题的添加方式：

  • 使用标题标签
  • 使用表格自带的标题标签caption
    • caption标签必须写在table开始标签的后面
  • 一个表格只能添加一对caption标签，只能有一个标签

表格中的一些固定嵌套

​ table 一般嵌套tr，tr中嵌套td

​ 只有td或th可以像div那样，作为容器嵌套任何东西

form标签

表单用来收集用户数据，并将其提交到对应的地址

我们的所有表单元素都要写在一个form标签内部，form标签是一个功能性的标签，不是结构性

form：是表单的语义化标签

属性：

​ name: 定义表单的名称

​ action：设置数据的提交的地址

​ method: 设置数据提交的方式

​ target： 页面打开方式

表单中的控件：textarea 文本域 select 下拉列表 input 输入框 button 按钮

textarea与label标签

textarea 多行文本域，多行文本输入框

属性:

​ rows 设置行高，属性值是数字

​ cols 设置列宽，属性值是数字 数字是字节

label 标签为表单控件（textarea，input）定义标注

for 规定label绑定到哪个表单控件（id）

​ 我们还可以将表单控件写在label标签内，实现联动

<label>简介 <textarea></textarea></label>

表单：action：提交数据的地址

​ method：提交数据的方式

​ target：在哪里查看结果

select标签

select 标签可以创建单选或多选菜单，属性：

​ multiple 设置多选

​ size 设置显示在页面上的列表的项数1-7

​ option 标签用定义下拉列表中的一个选项（一个条目），属性：

​ selected 设置默认被选中的列表项（布尔型属性）

内容只能是文本，不能出现标签

optgroup 标签定义选项组

label 为选项组规定描述

multiple 多选 复选

button标签

button 标签定义一个按钮，属性：

​ type 规定按钮的类型（button，reset，submint）

​ button控件与input type=“button”相比：

​ button标签之间的所有内容都是按钮的内容，比如文本或多媒体内容

兼容性：ie默认的类型是button，而其他浏览器中(包括w3c规范)的默认值是“submit“

input标签

表示输入的语义标签，用于收集用户信息，是单标签，也叫作自封闭标签

控件的类型由type属性确定：

​ text 单行文本框，文本输入框（默认的输入框类型）

​ pattern 规定输入字段的模式或格式（html5新增）

​ password 密码输入框

​ radio 单选框

​ checkbox 复选框，多选框

​ 单选和复选的name值必须一致，这样才能保证是同一个题干下面的不同选项

checked 设置默认被选中；布尔型属性

<input type="text"  id="username" pattern="[0-9]{2~6}">//2到5位数字
表单通用属性

value 定义送往服务器的选项值（表单控件的值）

autofocus 规定在页面加载后文本区域自动获得焦点

form 规定文本区域所属的一个或多个表单

name 规定文本区的名称

placeholder 规定描述文本区预期值的简短提示

disabled 规定禁用该文本区

readonly 规定文本区为只读

maxlength 规定文本区域的最大字符数（textarea与input）

required 规定文本区域是必填的（textarea与input）

input标签的type属性

image 图像域

src 设置图片的路径 width height设置图片的宽高

file 文件

hidden 隐藏文件

button 普通按钮

submit 提交按钮

reset 重置按钮

废弃标签

font 字体标签

frame 定义框架

b 加粗

i 斜体字

u 下划线

s 删除线

其他标签

pre 预格式化原样输出

code 单行代码

strong 强调标签，加粗效果

em 强调标签，斜体字效果

ins 下划线

iframe 内联框架

代码规范

标签名必须使用小写字母

属性名必须使用小写字母并用双引号包围属性值

使用4个空格作为一个缩进层级

class必须单词字母小写，单词以-分隔

id使用驼峰式命名
