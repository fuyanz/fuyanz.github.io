HTML

HTML语法规范

基本语法

  1. HTML标签是由尖括号包围的关键词，例如< html >
  2. HTML标签通常是成对出现的，例如< html >< /html >,我们把这么一对标签称为双标签。标签对中第一个标签是开始标签，第二个标签是结束标签。
  3. 有些特殊标签是单个标签。< img />,称为单标签。

标签关系

  1. 包含(父子)
<head>
    <title></title>
</head>

head是title的父

title是head的子

  1. 并列
<head></head>
<body></body>

head和body互为兄弟

HTML基本结构标签

第一个HTML网页

每个网页都会有一个基本的结构标签（也叫作骨架标签），页面的内容被书写在基本标签内。

html页面也被称为HTML文档。

<html>
    <head>
        <title>我的页面的标题</title>
    </head>
    <body>
        这是页面的内容书写位置
    </body>
</html>
标签名 定义 说明
< html>< /html> html标签 页面中最大的标签，被称为根标签
< head>< /head> 文档的头部 注意在head标签中我们必须设置的标签是title标签
< title>< /title> 文档的标题 让页面拥有一个属于自己的网页标题
< body>< /body> 文档的主体 元素包含文档的所有内容，页面内容基本都是放到body里

文档类型声明标签

**<!DOCTYPE>**文档类型声明，作用就是告诉浏览器使用哪种HTML版本来显示网页。

<!DOCPYTE html>
告诉浏览器当前页面采用html5版本来显示网页

声明位于文档最前端

不是一个HTML标签。就是一个文档类型声明标签。

lang语言种类

用于定义当前文档显示的语言。

  1. en定义语言为英语。 en英文网页

  2. zh-CN定义语言为中文。 zh-CN中文网页

字符集

字符集（Character set）是多个字符的集合。用来让计算机识别和存储各种文字。

通过 **< mata >**标签的 charset属性来规定HTML文档应该使用哪种字符编码。

<mata charset="UTF-8" />

charset常用的值有：GB2312、BIG5、GBK、UTF-8，UTF-8被称为万国码，基本包含了全世界所有国家需要用到的字符。

标签语义

标签的含义，标签用来干嘛的

在合适的地方根据标签的语义选择最合理的标签，可以让页面结构更清晰

标题标签< h1 >~< h6 >

6个等级的标题标签

head的缩写，头部；标题

作为标签使用，并根据重要性递减

一般网页只出现一个一级标签

特点

  1. 加了标题标签的文字会加粗，字号变大。
  2. 一个标签独占一行。

段落和换行标签

**< p >**标签用于定义段落。将文字分段显示。paragraph的缩写。

特点

  1. 文本在一个段落中会根据浏览器窗口的大小自动换行。
  2. 段落和段落之间有空隙。

**< br/>**强制换行某段文字。

在html中，一个段落中的文字会从左到右依次排列，直到浏览器窗口的右端，然后才自动换行。如果希望强制换行某段文字，就可以使用< br />

break 打断，换行。

特点

  1. < br/>是个单标签。
  2. < br/>只是简单的开始新的一行，和段落不一样，段落之间只会插入一些垂直的间距。

文本格式化标签

在网页中给一些文字设置特殊效果。

标签语义：突出重要性，比普通文字更重要。

语义 标签 说明
加粗 < strong>或< b> 推荐使用strong 语义更强烈
倾斜 < em>或< i> 推荐em
删除线 < del>或< s> 推荐del
下划线 < ins>或< u> 推荐ins

< div >和< span >标签

没有语义，就是一个盒子，用来装内容的。

div是divsion的缩写，分割；分区。

span 跨度、跨距。

特点

  1. div标签用来布局，在一行只能放置一个< div >大盒子

  2. span标签用来布局，一行可以放置多个< span > 小盒子

图像标签

图像标签：< img />单标签，用于定义html页面中的图像。

image的缩写：图像

< img src=”图像路径” />

src是img标签的必须属性，用于指定图像文件的路径和文件名。

属性：属于这个标签的特性。

其他属性：

属性 属性值
src 图片路径 必须属性
alt 文本 替换文本，图片不能显示时的文字
title 文本 提示文本，鼠标放到图片上显示的文本
width 像素 设置图像的宽度
height 像素 设置图像的高度
border 像素 设置图像的边框粗细

注意

  1. 图像标签可以拥有多个属性，必须写在标签名的后面
  2. 属性之间不分先后顺序，标签名与属性之间均以空格隔开
  3. 属性采取键值对的格式，key=”value”,属性=”属性值“

路径

目录文件夹和根目录：文件分类管理，放在一个文件夹

目录文件夹：普通文件夹

根目录：打开目录文件夹的第一层

相对路径：以引用文件所在位置为参考基础，而建立的目录路径。

同一级路径 符号
下一级路径 /
上一级路径 ../

从代码所在的位置出发，去寻找目标文件。

绝对路径：目录下的绝对位置，直接到达目标位置，就是盘符开始的路径。

超链接标签< a >

a标签用于定义超链接，从一个页面链接到另一个页面。

< a href=”跳转目标” target=”目标窗口的弹出方式”> 文本或图像< /a >

anchor 锚

属性 作用
href 指定链接目标的url地址，必须属性
target 指定链接页面的打开方式，_ self为默认值，_ blank在新窗口打开

链接分类

  1. 外部链接：如跳转到百度。
  2. 内部链接：网站内部页面之间的相互链接，直接链接内部页面名称。
  3. 空链接：#。
  4. 下载链接：href里面的地址是一个文件或者压缩包。
  5. 网页元素链接：在网页中的各种网页元素都可以添加超链接。
  6. 锚点链接：点击链接快速定位到页面中的某个位置。
    • 在链接文本的href属性中，设置属性值为 #名字的形式。
    • 在目标位置标签中添加id属性=刚才的名字。

注释标签

给代码人员便于理解和阅读，但不需要显示在页面。

< !— —>
